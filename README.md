# Day 1 - Google Form AI Email Responder
Automatically replies to Google Form submissions using Gemini AI.
## Tools: n8n · Google Forms · Google Sheets · Gemini API · Gmail
## How it works
1. Person fills Google Form
2. n8n detects new row in Google Sheet
3. Gemini AI writes a personalised reply
4. Gmail sends it automatically
